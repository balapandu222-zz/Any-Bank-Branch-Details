IFSC code

The Indian Financial System Code (familiarly known as IFSC) is an alphanumeric code that uniquely identifies a bank-branch participating in the two main electronic funds settlement systems in India: the real time gross settlement (RTGS) and the national electronic funds transfer (NEFT) systems.
